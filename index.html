// Wi-Fi Auth Log 기반 실시간 출퇴근 자동화 시스템 · 창원대학교 졸업논문

기존 출퇴근 시스템의 수동 인증 불편함을 해소하기 위해 설계한 Wi-Fi 연결 기반 자동 근태 추적 시스템. 사용자는 별도 앱 설치나 버튼 클릭 없이 AP에 연결·해제되는 것만으로 출퇴근이 자동 처리되며, Apache Kafka 실시간 스트리밍과 Node.js + Express 기반 Web UI로 관리자가 연결 해제 감지 0.7~0.8초 이내에 직원 근태를 실시간 모니터링할 수 있습니다.
